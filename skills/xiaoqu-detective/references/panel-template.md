# 11格漫画分镜标准模板 V1.0

## 标准结构（每话严格按11格来）

| 格子 | 功能定位 | 内容要求 |
|------|---------|---------|
| **第1格** | 封面 | 按封面SOP执行，大标题+全员登场 |
| **第2格** | 发现线索 | 逍逍发现异常，提出问题 |
| **第3格** | 爸爸瞎掰 | 阿杰用不靠谱的方式回答/类比 |
| **第4格** | 妈妈科普 | alla纠正爸爸，给出专业但通俗的解释 |
| **第5格** | 魔法展示 | 瑶瑶出手，用魔法展示看不见的原理 |
| **第6格** | 有趣实验 | 做一个小实验/观察，验证知识点 |
| **第7格** | 深入探索 | 更深入的知识点/内部结构展示 |
| **第8格** | 社会分工/细节 | 讲解更细的机制，或引出社会思考 |
| **第9格** | 残酷真相/深度 | 讲一点不那么美好但真实的事实 |
| **第10格** | 情感升华 | 联系人类生活，温暖治愈的总结 |
| **第11格** | 结尾+预告 | 暖心结尾，预告下一期主题 |

---

## 每格详细要求

### 第2格：发现线索
- **核心**：好奇心的起点
- **要素**：逍逍的震惊表情 + 发现的东西特写 + 一个大大的问号
- **台词**："等等！这是什么？！" "你们快来看！"

### 第3格：爸爸瞎掰
- **核心**：搞笑和共鸣
- **要素**：阿杰不靠谱的表情 + 似是而非的答案 + 最好能类比职场/人类社会
- **金句方向**："害，这不就是我们公司嘛！"
- **注意**：不能太离谱，要有点道理但又不完全对，给妈妈纠正留空间

### 第4格：妈妈科普
- **核心**：专业但通俗
- **要素**：alla温柔但专业的表情 + 准确的知识点
- **要求**：知识点不能错，要用小学生能听懂的话讲，不能有术语堆砌

### 第5格：魔法展示
- **核心**：视觉奇观
- **要素**：瑶瑶挥毛笔 + 发光的魔法效果 + 把看不见的东西可视化
- **示例**：信息素变成发光的线、地下巢穴变透明、声音变成波纹
- **要求**：画面一定要炫酷，有想象力

### 第6格：有趣实验
- **核心**：互动感和验证
- **要素**：亲手做一个小实验，观察变化
- **示例**：用手指划断蚂蚁路线、给蜗牛撒点水、用树叶逗鼠妇
- **作用**：让读者觉得"我也能做这个实验"

### 第7格：深入探索
- **核心**：更深入的知识点
- **要素**：展示更复杂的结构/机制
- **示例**：蚂蚁的社会分工、蜗牛的牙齿结构、鼠妇的鳃
- **要求**：比第4格的知识点更深一层，但依然要易懂

### 第8格：社会分工/细节
- **核心**：从生物到社会的连接点
- **要素**：把生物的行为和人类社会做类比
- **金句方向**："你别说，这和我们人类还真像"

### 第9格：残酷真相/深度
- **核心**：真实世界不只有美好
- **要素**：讲一点残酷但真实的自然法则
- **示例**：蚂蚁会抛弃受伤的同伴、蜗牛是农业害虫、鼠妇遇到危险会装死
- **作用**：增加深度，不只是低幼科普

### 第10格：情感升华
- **核心**：温暖治愈，价值观输出
- **要素**：一家人站在一起，夕阳，温暖的对话
- **金句方向**："慢一点也没关系啊" "每个生命都有它的意义"
- **要求**：不能说教，要自然，要真诚

### 第11格：结尾+预告
- **核心**：收尾和钩子
- **要素**：暖心的结尾画面 + 下期预告
- **预告格式**："下期预告：《XXX》—— [一句话悬念]"

---

## 台词分配原则

| 角色 | 台词占比 | 功能 |
|------|---------|------|
| 逍逍 | 30% | 提问、好奇、总结 |
| 阿杰 | 30% | 搞笑、瞎掰、人类社会类比 |
| alla | 30% | 科普、纠正、价值观输出 |
| 瑶瑶 | 5% | 魔法相关，话少但关键 |
| 媛媛 | 5% | 只有狗叫，不用说话 |

---

## 重要提醒

⚠️ **每话必须有：**
1.  一个准确的科普知识点
2.  至少一个爸爸的搞笑/共鸣点
3.  一次瑶瑶的魔法展示
4.  一个温暖治愈的结尾金句
5.  一个下期的预告钩子
