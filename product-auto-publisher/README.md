# 咸鱼全自动商品上架 + 自动发货 + 自动客服

自动从 ZIP 压缩包（夸克/百度网盘分享）中提取商品信息（主图+描述），自动发布到咸鱼，买家下单后自动发网盘链接，自动回复客户咨询。

完美适配卖**虚拟资料**的场景：你维护一个货源文档记录所有商品，自动批量上架，卖出自动发链接，咨询自动回复，完全全自动。

## 📋 功能特性

- ✅ **自动解析商品** - 自动解压 ZIP，识别主图（优先找"主图""封面"），提取标题、价格、描述
- ✅ **自动上架咸鱼** - 自动打开浏览器，上传图片，填写标题价格描述，你只需要选分类点发布
- ✅ **批量上架** - 支持从你的货源文档（CSV/Markdown/JSON）批量读取，自动逐个上架
- ✅ **自动发货** - 买家付款后，自动检测订单，自动发送网盘链接+提取码给买家
- ✅ **智能客服自动回复** - 内置常见售后问题回复（发货、链接、密码、退款等），支持自定义规则
- ✅ **持续监控** - 后台持续监控聊天消息，全自动处理不用管
- ✅ **商品持久化** - 所有商品保存在本地，重启自动加载
- ✅ **支持微信** - 同时支持微信朋友圈发布，生成文案整理图片

```bash
cd product-auto-publisher
pip install -r requirements.txt
```

必须安装：
- `selenium` - 浏览器自动化

## 🚀 使用方式

### 单个商品上架 + 启动全自动服务

```bash
python auto_service.py 你的商品.zip --share-url "https://pan.quark.cn/s/abc123" --share-code abcd --monitor
```

一条龙完成：
1. ✅ 解压 ZIP 解析商品信息（主图+标题+价格）
2. ✅ 打开浏览器进入咸鱼发布页
3. ✅ 自动上传所有图片
4. ✅ 自动填写标题、价格、描述
5. 👉 **你只需**：手动选分类 → 点击发布
6. ✅ 商品自动注册到自动发货列表
7. ✅ 开始自动监控聊天 → 自动回复咨询 → 下单自动发货

### 批量上架（从你的货源文档）

```bash
python batch_publish.py 你的货源.md --monitor
```

支持格式：**CSV** / **Markdown** / **JSON**，适合你"有一个网盘资料文档记录上架和货源地址"的场景。

示例 Markdown 格式：

```markdown
## photoshop 2024 全套教程
- 价格: 29.9
- 压缩包: ./files/ps2024教程.zip
- 链接: https://pan.quark.cn/s/abc123456
- 提取码: abcd
- 描述: 从入门到精通全套视频教程，包含素材文件。
```

程序自动读取所有商品，逐个帮你上架，完一个你点一下确认继续下一个。

### 只启动监控（已经上架完，只需要自动服务）

```bash
python auto_service.py --monitor
```

## 📁 文件组织

你只需要这样整理你的货源：

```
我的货源/
├── 商品目录.md         ← 这里记录所有商品
│   └── 每条商品对应：名称、价格、ZIP路径、网盘链接、提取码
├── files/
│   ├── 商品1.zip      ← ZIP 压缩包（里面有图片和文字介绍）
│   │   ├── 主图.jpg
│   │   ├── 图1.jpg
│   │   └── 介绍.txt
│   ├── 商品2.zip
│   └── ...
└── ...
```

然后一条命令：`python batch_publish.py 商品目录.md --monitor`，全部自动搞定！

### ZIP 压缩包内部格式

```
商品名称.zip
  ├── 主图.jpg       # 带"主图"会被优先选作主图
  ├── 细节图1.jpg
  ├── 细节图2.jpg
  ├── 细节图3.jpg
  └── 介绍.txt      # 第一行是标题，可以包含价格信息
```

`介绍.txt` 示例：

```
photoshop 2024 全套教程
价格: 29.9
原价: 59

从入门到精通全套视频教程，包含所有素材文件。
零基础也能学，永久更新！
```

程序自动提取标题、价格、描述。

## 🤖 自动回复功能

内置了这些常见问题的自动回复：

| 用户提问 | 自动回复内容 |
|---------|-------------|
| 什么时候发货？多久发货？ | 说明是自动发货，付款后链接自动发这里 |
| 链接呢？提取码呢？打不开 | 提示付款后自动发，要订单截图重发 |
| 解压密码是多少 | 说明密码在描述里，找不到会补发 |
| 文件损坏/打不开/解压失败 | 给出排查建议 |
| 可以退款吗？ | 说明虚拟商品一般不退，有问题核实处理 |
| 你好/在吗 | 友好问候 |

**添加自定义回复**，在 `config.json` 中配置：

```json
{
  "auto_reply": {
    "custom_rules": [
      {
        "keyword": "有没有货",
        "reply": "您好，有的，直接拍下付款就可以了，自动发货秒发链接😊",
        "regex": false
      },
      {
        "keyword": "可以便宜吗",
        "reply": "您好，价格已经很优惠了哦，不议价了呢😊",
        "regex": false
      }
    ]
  }
}
```

## ⚙️ 配置

复制 `config.example.json` 为 `config.json`，修改：

```json
{
  "download_dir": "downloads",
  "wechat": {
    "wechat_path": "C:\\Program Files (x86)\\Tencent\\WeChat\\WeChat.exe"
  },
  "xianyu": {
    "chrome_user_data_dir": "C:\\Users\\你的用户名\\AppData\\Local\\Google\\Chrome\\User Data"
  },
  "auto_reply": {
    "custom_rules": [
      // 在这里加你的自定义规则
    ]
  }
}
```

**重要前提**：
1. 安装 Chrome 浏览器
2. 下载对应版本的 `chromedriver.exe` 放到系统 PATH 里
3. 在 Chrome 里手动登录一次咸鱼，之后就可以保持登录了
4. 配置 `chrome_user_data_dir` 让浏览器复用已登录的会话
