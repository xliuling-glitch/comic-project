# 小区侦探科普漫画生成器

## 技能说明

这是「阿杰与alla阿拉」小区侦探科普漫画系列的专属创作技能包。
激活此技能后，AI将自动加载所有角色设定、格式规范、创作标准，确保输出风格统一、符合要求。

## 自动加载的资源

| 资源文件 | 内容 |
|---------|------|
| `references/characters.md` | 5个角色的完整人设设定 |
| `references/cover-sop.md` | 第一格封面标准SOP（必须遵守） |
| `references/panel-template.md` | 11格漫画分镜标准模板 |
| `references/style-guide.md` | 文风和内容创作指南 |

## 使用方式

### 基础使用
```
生成第N话，主题是XXX
```

### 完整输出（推荐）
```
用小区侦探技能，做蚂蚁那一期，输出完整全套内容
```

## 完整输出包含

1. ✅ 漫画主题和核心知识点
2. ✅ 11格完整漫画脚本（含台词、画面描述）
3. ✅ 第一格封面AI绘画提示词（严格按封面SOP）
4. ✅ 第2-11格分镜AI绘画提示词
5. ✅ 口播稿文案
6. ✅ 各平台发布文案

## 创作原则

1. **人设永不崩** - 严格按5个角色的设定写台词和行为
2. **科普要准确** - 生物知识点不能出错，通俗易懂
3. **笑点要自然** - 爸爸负责搞笑拆台，妈妈负责专业科普
4. **结尾有温度** - 最后要有情感升华，温暖治愈
5. **下期有预告** - 每话结尾预告下一期主题
