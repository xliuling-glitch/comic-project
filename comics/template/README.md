# 漫画创作模板

复制这个文件夹，改名为 `xxx-主题名` 就可以开始写新的一话了。

---

## 一、选题 Checklist

- [ ] 是小区里就能看到的东西/场景吗？
- [ ] 有1-2个准确的硬核知识点吗？
- [ ] 有可以类比人类社会/职场的笑点吗？
- [ ] 有温暖治愈的情感升华点吗？

---

## 二、11格标准结构

| 格子 | 功能 | 必须元素 |
|------|------|---------|
| 1 | 封面 | 全员登场 + 大标题 + 核心元素 |
| 2 | 发现线索 | 逍逍提问 |
| 3 | 爸爸瞎掰 | 职场/生活类比，制造笑点 |
| 4 | 妈妈科普 | 第一个核心知识点 |
| 5 | 魔法展示 | 瑶瑶出手，把看不见的东西可视化 |
| 6 | 有趣实验 | 读者看完也能亲手做的小实验 |
| 7 | 深入探索 | 第二个更深的知识点 |
| 8 | 社会类比 | 和人类社会/公司/生活联系 |
| 9 | 残酷真相 | 一点不那么美好但真实的事实 |
| 10 | 情感升华 | 温暖治愈的价值观输出 |
| 11 | 结尾+预告 | 暖心结尾，留下期钩子 |

---

## 三、台词分配比例

- 👦 逍逍：30% → 提问、好奇、总结
- 👨 阿杰：30% → 搞笑、瞎掰、人类社会类比
- 👩 alla：30% → 科普、纠正、价值观引导
- 🧚‍♀️ 瑶瑶：5% → 魔法相关，话少但关键
- 🐕 媛媛：5% → 只有狗叫，不用说话

---

## 四、金句公式

**简单的话 + 反常识的观点 + 温暖的治愈感**

例子：
- "不是每只蚂蚁都要当第一个找路的侦察兵"
- "慢一点也没关系啊，走点弯路也没关系啊"
- "睡着了不是偷懒，是大脑在升级"

---

## 五、封面标题风格优先级

1. **提问式**（点击率最高）：《你小时候一定也玩过吗？》
2. **共鸣式**：《原来我们都在排队往前走》
3. **好奇式**：《这个小东西居然这么厉害？》
