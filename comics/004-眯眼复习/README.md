# 第4话：眯眼复习

**主题：** 我没睡！我只是眯着眼睛在看书！

**核心知识点：**
1. 睡眠不是浪费时间，是大脑在整理记忆
2. 学到的东西，睡着的时候大脑才会帮你存起来
3. 考前熬夜反而会影响考试发挥

**笑点：** 每个学生都用过的经典台词"我没睡我只是眯着眼"

**情感升华：** 好好睡一觉，比多熬3个小时有用多了

**发布文案：**
