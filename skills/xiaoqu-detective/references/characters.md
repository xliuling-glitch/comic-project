# 「小区侦探」5个角色完整人设

---

## 👦 逍逍（儿子）- 主角视角

| 属性 | 设定 |
|------|------|
| **年龄** | 小学3-4年级 |
| **外貌** | 戴黑框眼镜，穿校服/运动服，脸上有雀斑，眼神充满好奇 |
| **性格** | 好奇心极强，爱提问，观察力敏锐，善良有同理心，认真但偶尔有点小较真 |
| **功能定位** | 
| | ✅ 故事的切入视角，代表孩子的眼光 |
| | ✅ 负责提出问题，推动剧情发展 |
| | ✅ 第一视角的观察者和记录者 |
| **口头禅** | "等等！这是什么？！" "为什么呀？" "太神奇了！" |
| **标志性动作** | 蹲在地上观察，手指着某个东西，眼睛睁得大大的 |

---

## 👨 阿杰（爸爸）- 普通人担当

| 属性 | 设定 |
|------|------|
| **年龄** | 35岁左右 |
| **外貌** | 戴眼镜，有点微胖，穿休闲装/羽绒服，头发有点乱，看起来就是普通的上班族 |
| **性格** | 幽默风趣，爱瞎掰，偶尔能说出有道理的话，大部分时候不靠谱，爱拿自己开涮，喜欢类比工作上的事 |
| **功能定位** |
| | ✅ 搞笑担当，负责拆台和冷笑话 |
| | ✅ 成年人视角，把生物知识类比成人类社会/职场 |
| | ✅ 说出普通人的困惑和共鸣 |
| **口头禅** | "害，这不就是我们公司嘛！" "我瞎猜的啊..." "你别说，还真有点像..." |
| **标志性动作** | 挠头，蹲在地上和儿子一起看，双手抱胸若有所思 |

---

## 👩 alla 阿拉（妈妈）- 科普担当

| 属性 | 设定 |
|------|------|
| **年龄** | 30多岁 |
| **外貌** | 温柔知性，扎马尾，穿休闲装，手里经常拿着浇花喷壶，园艺爱好者 |
| **性格** | 专业、靠谱、温柔但有原则，知识渊博，是家里的终极裁判，说的话大家都信 |
| **功能定位** |
| | ✅ 专业科普输出，知识点的权威来源 |
| | ✅ 纠正爸爸的瞎掰，给出正确答案 |
| | ✅ 价值观引导，温暖治愈的总结 |
| **口头禅** | "你爸又在瞎说了。" "其实啊..." "看到没，这就是..." |
| **标志性动作** | 蹲下来指着地面讲解，手里拿着喷壶，微笑着点头 |

---

## 🧚‍♀️ 瑶瑶（小仙女）- 魔法担当

| 属性 | 设定 |
|------|------|
| **外貌** | 国风打扮，穿汉服/古风裙子，手里永远拿着一支巨大的毛笔，会飘在空中 |
| **性格** | 神秘，话不多但一出手就是大招，有点呆萌，喜欢用魔法展示看不见的东西 |
| **功能定位** |
| | ✅ 用魔法把看不见的东西可视化（信息素、地下巢穴、细胞结构等） |
| | ✅ 推动剧情的超自然力量，解决人类视角看不到的问题 |
| | ✅ 增加漫画的魔幻感和想象力 |
| **口头禅** | "看我的~" "这样你们就看懂了~" |
| **标志性动作** | 挥毛笔，周围出现发光的魔法特效，飘在半空中 |

---

## 🐕 媛媛（中华田园犬）- 萌宠担当

| 属性 | 设定 |
|------|------|
| **外貌** | 四眼铁包金中华田园犬，黑背黄脸，眉毛上有两个小黄点（四眼），尾巴总是摇着 |
| **性格** | 活泼、好奇、贪吃、忠诚，偶尔闯祸，大部分时候很乖，用鼻子发现线索 |
| **功能定位** |
| | ✅ 卖萌担当，负责可爱，缓和严肃的科普氛围 |
| | ✅ 用鼻子发现线索，推动剧情（闻到奇怪的味道，找到虫子等） |
| | ✅ 偶尔闯祸制造笑点（踩散蚂蚁队伍，打翻花盆等） |
| **口头禅** | "汪！" "呜~"（只有狗叫，不用说话） |
| **标志性动作** | 摇尾巴，用鼻子到处闻，吐舌头笑，歪头 |

## 角色互动关系图

```
逍逍（提问）→ alla妈妈（解答）
     ↓                ↓
阿杰爸爸（瞎掰/搞笑补充）
     ↓
瑶瑶（魔法展示）
     ↓
媛媛（发现线索/闯祸）
```

## 重要原则

⚠️ **人设永不崩！** 每个角色的台词和行为必须符合设定，不能让妈妈说搞笑的话，不能让爸爸讲专业科普，不能让瑶瑶像普通人一样说话。
