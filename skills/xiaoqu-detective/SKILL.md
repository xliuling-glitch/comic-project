# 小区侦探科普漫画生成器

## 技能说明

这是「阿杰与alla阿拉」小区侦探科普漫画系列的专属创作技能包。
激活此技能后，AI将自动加载所有角色设定、格式规范、创作标准，确保输出风格统一、符合要求。

## 自动加载的资源

| 资源文件 | 内容 |
|---------|------|
| `references/characters.md` | 5个角色的完整人设设定 |
| `references/cover-sop.md` | 第一格封面标准SOP（必须遵守，仅科普篇用） |
| `references/panel-template.md` | 科普篇：11格漫画分镜标准模板 |
| `references/story-template.md` | **故事篇：成长/亲子/情感向自由格式模板** |
| `references/style-guide.md` | 文风和内容创作指南 |
| `references/spider-output-template.md` | 蜘蛛篇发布文案模板（含爆款标题+各平台文案+金句） |
| `references/运营SOP.md` | 封面统一标准+关注引导+评论区运营+发布自检清单 |

> 💡 **发布文案模板参考**：`spider-output-template.md` 以蜘蛛篇为范本，展示了完整的发布输出规范。每期生成时按此格式输出爆款标题合集（抖音/小红书/公众号）、各平台完整文案、结尾金句集。每期需根据主题特点和平台调性定制，不可直接照搬措辞。

## 两套模板体系

| | 科普篇 | 故事篇 |
|:---|:---|:---|
| 用途 | 昆虫/自然/科学知识科普 | 成长故事/亲子冲突/家庭情感 |
| 格子数 | 严格11格 | 8-15格，不固定 |
| 格子大小 | 基本均匀 | 情绪驱动，可大可小 |
| 节奏 | 知识点顺序输出 | 情绪弧线：铺垫→冲突→爆发→转折→收尾 |
| 结尾 | 温暖金句+下期预告 | 余味为主，不一定甜 |
| 封面 | 按封面SOP统一格式 | 可以不一样，按故事氛围设计 |
| **看哪个模板** | `panel-template.md` | `story-template.md` |

## 选题原则

选题核心逻辑：**「生活场景 × 受众群体」= 选题**

- 不要为了科普而科普
- 先问：这个季节，小区里的人正在高频经历什么？
- 再问：这个经历能不能延伸成有共鸣的内容？

### 选题来源
1. **场景驱动** — 大家正在经历的（被蚊子咬、听蝉叫、看到飞蚁扑灯）
2. **情绪驱动** — 大家心里有但没说出口的（被否定、坚持自己、家庭矛盾）
3. **复刻爆款模板** — 提取爆款内容的叙事结构，装进自己世界观的场景

## 使用方式

```
用小区侦探技能，做蚊子那一期，输出完整全套内容
```

### 故事篇
```

```

## 科普篇完整输出包含

1. ✅ 漫画主题和核心知识点
2. ✅ 11格完整漫画脚本（含台词、画面描述）
3. ✅ 第一格封面AI绘画提示词（严格按封面SOP）
4. ✅ 第2-11格分镜AI绘画提示词
5. ✅ 爆款标题合集（抖音/视频号、小红书、公众号，至少各2条）
6. ✅ 各平台完整文案（抖音口播稿、小红书文案、朋友圈短文案）
7. ✅ 结尾金句集（至少5条，配最后一帧画面）
8. ✅ 下期预告

## 故事篇完整输出包含

1. ✅ 故事主题和情绪定位
2. ✅ 自由格式的分镜脚本（格子数/大小按情绪走）
3. ✅ 封面设计方向（不严格按科普篇SOP）
4. ✅ 每格AI绘画提示词（含情绪标记）
5. ✅ 爆款标题合集
6. ✅ 各平台完整文案
7. ✅ 结尾金句
8. ✅ 可选：下期预告

## 创作原则（通用）

1. **人设可以突破** - 科普篇严格按角色功能；故事篇角色可以更真实（爸爸可以严肃、逍逍可以发脾气）
2. **科普要准确** - 科普篇知识点不能出错
3. **情绪要真实** - 故事篇不强行搞笑，不强行大团圆
4. **结尾有余味** - 余味比甜味久，不一定非要下期预告
5. **选对模板** - 科普内容用科普模板，故事内容用故事模板
