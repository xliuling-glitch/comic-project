# 第3话：蜗牛

**主题：** 慢一点也没关系啊

**核心知识点：**
1. 蜗牛有两万多颗牙齿，是世界上牙齿最多的动物
2. 蜗牛的粘液可以修复伤口，现在已经做成了护肤品
3. 蜗牛冬天会冬眠，不吃不喝睡几个月

**笑点：** 爸爸把蜗牛的慢类比成"周五下午摸鱼的我"
