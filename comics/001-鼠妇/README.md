# 第1话：鼠妇

**主题：** 你小时候一定玩过的小虫子，居然和虾是亲戚？

**核心知识点：**
1. 鼠妇不是昆虫，是甲壳动物，和虾、螃蟹是亲戚
2. 鼠妇用鳃呼吸，所以必须生活在潮湿的地方
3. 鼠妇吃腐殖质，是大自然的清洁工

**笑点：** 爸爸把鼠妇球类比成"公司团建大家抱成一团取暖"

**情感升华：** 每个小生命都有它存在的意义，哪怕是不起眼的小虫子
