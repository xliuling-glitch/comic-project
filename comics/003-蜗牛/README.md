# 第3话：蜗牛

**标题：** 蜗牛壳为什么都是顺时针的？

**核心知识点：**
1. 蜗牛壳有左旋和右旋之分，大多数是右旋（顺时针），少数天生左旋
2. 壳的方向是基因决定的，一出生就定了，改不了
3. 左旋和右旋的蜗牛没法在一起——生殖孔位置相反，对不上
4. 这叫"手性"（chirality），人体也有：心脏偏左、肝脏在右

**笑点：** 爸爸把蜗牛壳比作"螺丝"，说左旋右旋拧不到一起

**情感升华：** 方向没有对错，每个人都有自己天生的样子，做自己就好
