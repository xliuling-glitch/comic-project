# 第2话：蚂蚁

**主题：** 你有没有觉得我们越活越像蚂蚁？

**核心知识点：**
1. 蚂蚁用信息素交流，走过的路会留下气味标记
2. 蚂蚁社会分工：蚁后、工蚁、兵蚁、雄蚁
3. 蚁后能活十几年，工蚁只能活几个月

**笑点：** 爸爸把蚂蚁排队类比成"早高峰挤地铁的打工人"

**情感升华：** 不是每只蚂蚁都要当侦察兵，普普通通的工蚁，也在认真地活着

**发布文案：**
```
🐜 你有没有蹲下来看过蚂蚁排队？
原来它们和我们人类一模一样！

有侦察兵、有打工人、甚至还有专门生孩子的"全职妈妈"
原来我们每天早高峰挤地铁的样子
和蚂蚁排队搬食物，根本没有区别

今天，向每一只认真搬砖的小蚂蚁致敬
也向每一个认真生活的你致敬✨

#亲子 #科普 #打工人 #漫画
```
