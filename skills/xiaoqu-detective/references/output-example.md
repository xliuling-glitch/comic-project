## 二、完整示例（第1话鼠妇篇片段）

```
# 第1话：《你小时候一定也玩过西瓜虫吗？》

## 📋 本话核心知识点
1.  鼠妇不是昆虫，是甲壳动物，和虾是亲戚
2.  鼠妇用鳃呼吸，所以必须生活在潮湿的地方
3.  遇到危险会卷成球，这是它们的防御机制

---

## 🎨 第1格封面
**画面描述**：
小区3单元门口，逍逍蹲在最前面震惊地看着地上，阿杰和alla站在他身后，瑶瑶飘在右上角拿着大毛笔，媛媛摇着尾巴站在旁边。地上有几只鼠妇。画面底部有黑色半透明遮罩。

```

```
...以此类推
```

## 三、格式要求

1.  **标题必须用 # 号分级**，方便阅读
2.  **AI绘画提示词必须放在代码块** 里（```包裹），方便复制
3.  **每格漫画的画面描述要详细**，能直接用来生成图片
4.  **口播稿要口语化**，像聊天一样，不要书面语
5.  **各平台文案要区分长度和风格**，不能一稿多投用一样的
