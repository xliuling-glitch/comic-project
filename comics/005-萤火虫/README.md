# 第5话：萤火虫

**主题：** 你有多久没见过萤火虫了？

**核心知识点：**
1. 萤火虫发光是为了找对象，每种萤火虫都有自己的闪灯密码
2. 萤火虫幼虫是肉食动物，专门吃蜗牛
3. 成虫只能活7天左右，而且只喝水不吃饭

**笑点：** 爸爸把萤火虫发光类比成"刷交友App发动态，闪得越亮越受欢迎"

**情感升华：** 城市里的萤火虫越来越少了，不是它们消失了，是我们走得太快，忘了停下来看一眼
