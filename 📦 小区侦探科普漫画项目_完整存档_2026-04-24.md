## 🐜 第1话：鼠妇（西瓜虫）科普

### 文章主题
我的书桌下，藏着一片迷你森林——鼠妇养成记

### 核心知识点
1.  **鼠妇不是昆虫** - 属于甲壳纲动物，和虾是近亲
2.  **用鳃呼吸** - 所以必须生活在潮湿环境中
3.  **装死防御** - 受到惊吓会卷成完美的小球
4.  **社会行为** - 永远三五成群，一起吃饭一起行动
5.  **妈妈带娃** - 幼崽会跟着妈妈活动

### 饲养教程
- **容器**：透明外卖盒即可
- **垫料**：撕碎的落叶 + 树皮
- **食物**：苹果核、黄瓜皮、枯叶（一周喂一次）
- **环境**：书桌抽屉里（避光、潮湿）
- **注意**：绝对不能晒太阳！

### 文章金句
> 原来我们的身边就藏着这么多奇妙的小生命，不需要去动物园，不需要去远方，只要你弯下腰，翻开一块石头，就能发现一整个奇妙的世界。

## 🐜 第2话：蚂蚁科普

### 文章主题
为什么服务行业招人难，蚂蚁也留不住（社会性昆虫的职场启示）

### 核心知识点
1.  **蚁后vs工蚁** - 阶级分化和寿命差异
2.  **信息素交流** - 蚂蚁的"语言"系统
3.  **行军路径** - 为什么蚂蚁永远走直线
4.  **社会分工** - 保育蚁、工蚁、兵蚁的不同职责
5.  **巢穴系统** - 复杂的地下城市结构

### 核心洞察
> 蚂蚁的社会和人类的服务行业惊人相似：
> - 都是高强度、高压力的工作
> - 都存在严重的人员流失问题
> - 都需要系统性的解决方案
> - 都证明了：只靠"招人"解决不了"留人"的问题

## 🖼️ 抖音爆款封面设计方案

### 方案A：反差共鸣型（推荐）
- **主标题**：我在书桌下 / 养了一窝虫子
- **副标题**：我妈说我疯了，但老师都夸我
- **视觉**：你蹲在地上翻石头的真实照片 + 左下角鼠妇特写小图
- **配色**：黑底白字，标题用黄色描边突出

### 方案B：好奇提问型
- **主标题**：你小时候 / 玩过这种虫子吗？
- **小字**：90%的人都不知道它有多酷
- **视觉**：模糊的小区花园背景 + 中间清晰的鼠妇特写
- **配色**：绿色自然背景，白色粗体字

### 方案C：价值干货型
- **主标题**：这可能是 / 小学生能养的 / 最酷宠物
- **列表式**：✅ 不咬人 ✅ 不花钱 ✅ 不会死
- **视觉**：透明盒子里一整窝鼠妇的俯拍
- **额外信息**：底部加"零成本，看完就会养"

## 💾 OpenClaw 本地数据备份指南

### 备份位置
```
Windows系统：
C:\Users\你的用户名\.openclaw

快捷跳转：
在文件资源管理器地址栏输入 %USERPROFILE%\.openclaw
```

### 备份内容说明
| 文件/文件夹 | 说明 |
|------------|------|
| `database.sqlite` | 核心数据库，所有聊天记录、生成的文案、提示词、历史对话 |
| `media/` | 所有上传和生成的图片、音频、视频素材 |
| `config/` | 所有配置、API密钥、插件设置、自定义参数 |
| `extensions/` | 所有安装的插件和技能 |

### 备份建议
1.  **完整备份**：直接复制整个 `.openclaw` 文件夹
2.  **备份频率**：每周备份一次，尤其是重要内容更新后
3.  **换电脑恢复**：将备份的 `.openclaw` 文件夹复制到新电脑的相同位置即可
